Basic Application!

This is a very basic application using AngularJS, Bootstrap, and XAMPP which includes PHP, Apache, and MySql. I built this very quickly just to show how to use PHP as an API with AngularJS as the frontend in order to avoid rendering HTML on the server. The search box to the upper right is designed to allow a quick search of the data below.
